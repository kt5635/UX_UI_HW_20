Yard Sale Trail

Overview

The goal of this project was create a yard sale app that will help buyers looking for second hand items, find information about specific items for sale at local yard sales. We took this project from an idea and turned it into a mid-fidelity prototype

Team: Team of four UX designers Responsibilities:

  • User research
  • User persona
  • Ideation
  • Wireframing
  • User testing

Tools: FigJam, Figma, Balsamiq
